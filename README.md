# 个人文件管理体系

> 让AI帮你管理桌面文件，每天只专注三件事：计划、执行、复盘。

## 这是什么？

一个基于AI的桌面文件管理方案，帮助你：
- 📁 自动分类文件到正确位置
- 📝 每日计划（最多3件事）
- 🔄 每日复盘
- 📊 周总结
- ☁️ 自动备份到GitHub

## 快速开始

### 准备工作

1. 安装 [Git](https://git-scm.com/)
2. 注册 [GitHub](https://github.com) 账号
3. 创建一个GitHub仓库

### 使用方法

与AI对话时，说出触发词：

| 触发词 | 功能 |
|-------|------|
| 早上好 | 写今日计划 |
| 你好 | 整理文件 |
| 晚安 | 写每日复盘 |
| 周总结 | 写周复盘 |

### 文件夹结构

```
桌面/
├── 马哲/        ← 指导思想
├── 知识库/      ← 日记、笔记
├── AIGC/        ← 代码项目
├── 自媒体/      ← 视频相关
└── 仓库/        ← 杂物
```

## 核心理念

**想写的放知识库，做的放AIGC/自媒体，杂物放仓库。**
