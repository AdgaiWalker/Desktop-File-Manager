# 个人文件管理体系

> 让AI帮你管理桌面文件，每天只专注三件事：计划、执行、复盘。

## 这是什么？

一个基于AI的桌面文件管理方案，帮助你：
- 📁 自动分类文件到正确位置
- 📝 每日计划（最多3件事）
- 🔄 每日复盘
- 📊 周总结
- ☁️ 自动备份到GitHub

## 使用前提

⚠️ **第一次使用前，请先截图桌面留底，备份到GitHub仓库。**

防止AI误删或丢失文件。

## 快速开始

### 准备工作

1. 安装 [Git](https://git-scm.com/)
2. 注册 [GitHub](https://github.com) 账号
3. 创建一个GitHub仓库
4. **截图当前桌面，备份留底**

### 使用方法

**AI主动提醒：** 每次对话开始时，AI会自动检测并提醒你：

| 时间段 | 状态 | AI行为 |
|-------|------|-------|
| 早上(5-12点) | 无计划 | 主动问："早上好！今天要写计划吗？" |
| 下午(12-18点) | 无计划 | "下午好！今天还没写计划，要补上吗？" |
| 晚上(18-24点) | 无复盘 | 主动问："晚上好！今天要写复盘吗？" |

**触发词：**

| 触发词 | 功能 |
|-------|------|
| 早上好 | 写今日计划 |
| 你好 | 整理文件 |
| 晚安 | 写每日复盘 |
| 周总结 | 写周复盘 |

### 文件夹结构

```
桌面/
├── 马哲/        ← 指导思想
├── 知识库/      ← 日记、笔记
├── AIGC/        ← 代码项目
├── 自媒体/      ← 视频相关
└── 仓库/        ← 杂物
```

## 核心理念

**想写的放知识库，做的放AIGC/自媒体，杂物放仓库。**

## 常见问题

### 1. GitHub仓库名main与git本地仓库名master不同怎么办？

初次在桌面git建立仓库，本地仓库默认master。

**解决方案：**
```
提示词：我的GitHub仓库名是main，将本地仓库名修改为main
```

或手动执行：
```bash
git branch -m master main
git push -u origin main
```

### 2. AI误删了文件怎么办？

从GitHub仓库恢复：
```bash
git restore 文件名
```

或从远程拉取：
```bash
git fetch origin
git reset --hard origin/main
```

### 3. 文件太多，整理太累怎么办？

分批处理，每天处理一部分。AI会记住进度，下次继续。
